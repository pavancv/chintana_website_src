ध्यानयोग/आत्मसंयमयोग

श्लोक ६.१

श्री भगवानुवाच
अनाश्रितः कर्मफलं कार्यं कर्म करोति यः।
स संन्यासी च योगी च न निरग्निर्न चाक्रियः ॥ ६.१ ॥

टीक / टिप्पणि

श्लोक ६.२

यं संन्यासमिति प्राहुर्योगं तं विद्धि पाण्डव।
न ह्यसंन्यस्तसङ्कल्पो योगी भवति कश्चन ॥ ६.२॥

टीक / टिप्पणि

श्लोक ६.३

आरुरुक्षोर्मुनेर्योगं कर्म कारणमुच्यते।
योगारूढस्य तस्यैव शमः कारणमुच्यते ॥६.३॥

टीक / टिप्पणि

श्लोक ६.४

यदा हि नेन्द्रियार्थेषु न कर्मस्वनुषज्जते।
सर्वसङ्कल्पसंन्यासी योगारूढस्तदोच्यते॥६.४॥

टीक / टिप्पणि

श्लोक ६.५

उद्धरेदात्मनाऽऽत्मानं नात्मानमवसादयेत्।
आत्मैव ह्यात्मनो बन्धुरात्मैव रिपुरात्मनः॥६.५॥

टीक / टिप्पणि

श्लोक ६.६

बन्धुरात्माऽऽत्मनस्तस्य येनात्मैवात्मना जितः।
अनात्मनस्तु शत्रुत्वे वर्तेतात्मैव शत्रुवत् ॥ ६.६ ॥

टीक / टिप्पणि

श्लोक ६.७

जितात्मनः प्रशान्तस्य परमात्मा समाहितः।
शीतोष्णसुखदुःखेषु तथा मानापमानयोः ॥ ६.७ ॥

टीक / टिप्पणि

श्लोक ६.८

ज्ञानविज्ञानतृप्तात्मा कूटस्थो विजितेन्द्रियः।
युक्त इत्युच्यते योगी समलोष्टाश्मकाञ्चनः॥६.८॥

टीक / टिप्पणि

श्लोक ६.९

सुहृन्मित्रार्युदासीनमध्यस्थद्वेष्यबन्धुषु।
साधुष्वपि च पापेषु समबुद्धिर्विशिष्यते॥६.९॥

टीक / टिप्पणि

श्लोक ६.१०

योगी युञ्जीत सततमात्मानं रहसि स्थितः।
एकाकी यतचित्तात्मा निराशीरपरिग्रहः॥६.१०॥

टीक / टिप्पणि

श्लोक ६.११

शुचौ देशे प्रतिष्ठाप्य स्थिरमासनमात्मनः।
नात्युच्छ्रितं नातिनीचं चैलाजिनकुशोत्तरम्॥६.११॥

टीक / टिप्पणि

श्लोक ६.१२

तत्रैकाग्रं मनः कृत्वा यतचित्तेन्द्रियक्रियः।
उपविश्यासने युञ्ज्याद्योगमात्मविशुद्धये॥६.१२॥

टीक / टिप्पणि

श्लोक ६.१३

समं कायशिरोग्रीवं धारयन्नचलं स्थिरः।
संप्रेक्ष्य नासिकाग्रं स्वं दिशश्चानवलोकयन्॥६.१३॥

टीक / टिप्पणि

श्लोक ६.१४

प्रशान्तात्मा विगतभीर्ब्रह्मचारिव्रते स्थितः।
मनः संयम्य मच्चित्तो युक्त आसीत मत्परः॥६.१४॥

टीक / टिप्पणि

श्लोक ६.१५

युञ्जन्नेवं सदाऽऽत्मानं योगी नियतमानसः।
शान्तिं निर्वाणपरमां मत्संस्थामधिगच्छति॥६.१५॥

टीक / टिप्पणि

श्लोक ६.१६

नात्यश्नतस्तु योगोऽस्ति न चैकान्तमनश्नतः।
न चातिस्वप्नशीलस्य जाग्रतो नैव चार्जुन॥६.१६॥

टीक / टिप्पणि

श्लोक ६.१७

युक्ताहारविहारस्य युक्तचेष्टस्य कर्मसु।
युक्तस्वप्नावबोधस्य योगो भवति दुःखहा॥६.१७॥

टीक / टिप्पणि

श्लोक ६.१८

यदा विनियतं चित्तमात्मन्येवावतिष्ठते।
निःस्पृहः सर्वकामेभ्यो युक्त इत्युच्यते तदा॥६.१८॥

टीक / टिप्पणि

श्लोक ६.१९

यथा दीपो निवातस्थो नेङ्गते सोपमा स्मृता।
योगिनो यतचित्तस्य युञ्जतो योगमात्मनः॥६.१९॥

टीक / टिप्पणि

श्लोक ६.२०

यत्रोपरमते चित्तं निरुद्धं योगसेवया।
यत्र चैवात्मनाऽऽत्मानं पश्यन्नात्मनि तुष्यति॥६.२०॥

टीक / टिप्पणि

श्लोक ६.२१

सुखमात्यन्तिकं यत्तद्बुद्धिग्राह्यमतीन्द्रियम्।
वेत्ति यत्र न चैवायं स्थितश्चलति तत्त्वतः॥६.२१॥

टीक / टिप्पणि

श्लोक ६.२२

यं लब्ध्वा चापरं लाभं मन्यते नाधिकं ततः।
यस्मिन्स्थितो न दुःखेन गुरुणापि विचाल्यते॥६.२२॥

टीक / टिप्पणि

श्लोक ६.२३

तं विद्याद् दुःखसंयोगवियोगं योगसंज्ञितम्।
स निश्चयेन योक्तव्यो योगोऽनिर्विण्णचेतसा॥६.२३॥

टीक / टिप्पणि

श्लोक ६.२४

सङ्कल्पप्रभवान्कामांस्त्यक्त्वा सर्वानशेषतः।
मनसैवेन्द्रियग्रामं विनियम्य समन्ततः॥६.२४॥

टीक / टिप्पणि

श्लोक ६.२५

शनैः शनैरुपरमेद् बुद्ध्या धृतिगृहीतया।
आत्मसंस्थं मनः कृत्वा न किञ्चिदपि चिन्तयेत्॥६.२५॥

टीक / टिप्पणि

श्लोक ६.२६

यतो यतो निश्चरति मनश्चञ्चलमस्थिरम्।
ततस्ततो नियम्यैतदात्मन्येव वशं नयेत्॥६.२६॥

टीक / टिप्पणि

श्लोक ६.२७

प्रशान्तमनसं ह्येनं योगिनं सुखमुत्तमम्।
उपैति शान्तरजसं ब्रह्मभूतमकल्मषम्॥६.२७॥

टीक / टिप्पणि

श्लोक ६.२८

युञ्जन्नेवं सदाऽऽत्मानं योगी विगतकल्मषः।
सुखेन ब्रह्मसंस्पर्शमत्यन्तं सुखमश्नुते॥६.२८॥

टीक / टिप्पणि

श्लोक ६.२९

सर्वभूतस्थमात्मानं सर्वभूतानि चात्मनि।
ईक्षते योगयुक्तात्मा सर्वत्र समदर्शनः॥६.२९॥

टीक / टिप्पणि

श्लोक ६.३०

यो मां पश्यति सर्वत्र सर्वं च मयि पश्यति।
तस्याहं न प्रणश्यामि स च मे न प्रणश्यति॥६.३०॥

टीक / टिप्पणि

श्लोक ६.३१

सर्वभूतस्थितं यो मां भजत्येकत्वमास्थितः।
सर्वथा वर्तमानोऽपि स योगी मयि वर्तते॥६.३१॥

टीक / टिप्पणि

श्लोक ६.३२

आत्मौपम्येन सर्वत्र समं पश्यति योऽर्जुन।
सुखं वा यदि वा दुःखं सः योगी परमो मतः॥६.३२॥

टीक / टिप्पणि

श्लोक ६.३३

अर्जुन उवाच
योऽयं योगस्त्वया प्रोक्तः साम्येन मधुसूदन।
एतस्याहं न पश्यामि चञ्चलत्वात् स्थितिं स्थिराम्॥६.३३॥

टीक / टिप्पणि

श्लोक ६.३४

चञ्चलं हि मनः कृष्ण प्रमाथि बलवद्दृढम्।
तस्याहं निग्रहं मन्ये वायोरिव सुदुष्करम्॥६.३४॥

टीक / टिप्पणि

श्लोक ६.३५

श्री भगवानुवाच
असंशयं महाबाहो मनो दुर्निग्रहं चलं।
अभ्यासेन तु कौन्तेय वैराग्येण च गृह्यते॥६.३५॥

टीक / टिप्पणि

श्लोक ६.३६

असंयतात्मना योगो दुष्प्राप इति मे मतिः।
वश्यात्मना तु यतता शक्योऽवाप्तुमुपायतः॥६.३६॥

टीक / टिप्पणि

श्लोक ६.३७

अर्जुन उवाच
अयतिः श्रद्धयोपेतो योगाच्चलितमानसः।
अप्राप्य योगसंसिद्धिं कां गतिं कृष्ण गच्छति॥६.३७॥

टीक / टिप्पणि

श्लोक ६.३८

कच्चिन्नोभयविभ्रष्टश्छिन्नाभ्रमिव नश्यति।
अप्रतिष्ठो महाबाहो विमूढो ब्रह्मणः पथि॥६.३८॥

टीक / टिप्पणि

श्लोक ६.३९

एतन्मे संशयं कृष्ण छेत्तुमर्हस्यशेषतः।
त्वदन्यः संशयस्यास्य छेत्ता न ह्युपपद्यते॥६.३९॥

टीक / टिप्पणि

श्लोक ६.४०

श्री भगवानुवाच
पार्थ नैवेह नामुत्र विनाशस्तस्य विद्यते।
नहि कल्याणकृत्कश्चिद्दुर्गतिं तात गच्छति॥६.४०॥

टीक / टिप्पणि

श्लोक ६.४१

प्राप्य पुण्यकृतां लोकानुषित्वा शाश्वतीः समाः।
शुचीनां श्रीमतां गेहे योगभ्रष्टोऽभिजायते॥६.४१॥

टीक / टिप्पणि

श्लोक ६.४२

अथवा योगिनामेव कुले भवति धीमताम् ।
एतद्धि दुर्लभतरं लोके जन्म यदीदृशम् ॥६.४२॥

टीक / टिप्पणि

श्लोक ६.४३

तत्र तं बुद्धिसंयोगं लभते पौर्वदेहिकम्।
यतते च ततो भूयः संसिद्धौ कुरुनन्दन॥६.४३॥

टीक / टिप्पणि

श्लोक ६.४४

पूर्वाभ्यासेन तेनैव ह्रियते ह्यवशोऽपि सः।
जिज्ञासुरपि योगस्य शब्दब्रह्मातिवर्तते॥६.४४॥

टीक / टिप्पणि

श्लोक ६.४५

प्रयत्नाद्यतमानस्तु योगी संशुद्धकिल्बिषः।
अनेकजन्मसंसिद्धस्ततो याति परां गतिम्॥६.४५॥

टीक / टिप्पणि

श्लोक ६.४६

तपस्विभ्योऽधिको योगी ज्ञानिभ्योऽपि मतोऽधिकः।
कर्मिभ्यश्चाधिको योगी तस्माद्योगी भवार्जुन॥६.४६॥

टीक / टिप्पणि

श्लोक ६.४७

योगिनामपि सर्वेषां मद्गतेनान्तरात्मना।
श्रद्धावान्भजते यो मां स मे युक्ततमो मतः॥६.४७॥

टीक / टिप्पणि


ॐ तत्सदिति श्रीमद्गगवद्वीतासूपनिषत्सु ब्रह्मविद्यायां योगशास्त्रे श्रीकृष्णार्जुनसंवादे आत्मसंयमयोगो नाम षष्ठोऽध्यायः ॥ ६ ॥

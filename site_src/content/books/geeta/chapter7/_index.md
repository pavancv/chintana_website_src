# ज्ञानविज्ञानयोग {class="text-center"}

---

## श्लोक ७.१ {class="visually-hidden"}

{{< shloka  link="Shloka1" >}}

**श्री भगवानुवाच**  
मय्यासक्तमनाः पार्थ योगं युञ्जन्मदाश्रयः ।   
असंशयं समग्रं मां यथा ज्ञास्यसि तच्छृणु ॥७.१॥

{{< /shloka >}}


## श्लोक ७.२ {class="visually-hidden"}
{{< shloka  link="Shloka2" >}}

ज्ञानं तेऽहं सविज्ञानमिदं वक्ष्याम्यशेषतः ।  
यज्ज्ञात्वा नेह भूयोऽन्यज्ज्ञातव्यमवशिष्यते॥७.२॥

{{< /shloka >}}


## श्लोक ७.३ {class="visually-hidden"}
{{< shloka  link="Shloka3" >}}

मनुष्याणां सहस्रेषु कश्चिद्यतति सिद्धये ।  
यततामपि सिद्धानां कश्चिन्मां वेत्ति तत्त्वतः ॥७.३॥

{{< /shloka >}}


## श्लोक ७.४ {class="visually-hidden"}

{{< shloka  link="Shloka4" >}}

भूमिरापोऽनलो वायुः खं मनो बुद्धिरेव च ।  
अहङ्कार इतीयं मे भिन्ना प्रकृतिरष्टधा ॥७.४॥

{{< /shloka >}}


## श्लोक ७.५ {class="visually-hidden"}

{{< shloka  link="Shloka5" >}}

अपरेयमितस्त्वन्यां प्रकृतिं विद्धि मे पराम्।  
जीवभूतां महाबाहो ययेदं धार्यते जगत् ॥७.५ ॥

{{< /shloka >}}


## श्लोक ७.६ {class="visually-hidden"}

{{< shloka  link="Shloka6" >}}

एतद्योनीनि भूतानि सर्वाणीत्युपधारय ।  
अहं कृत्स्नस्य जगतः प्रभवः प्रलयस्तथा॥७.६॥

{{< /shloka >}}


## श्लोक ७.७ {class="visually-hidden"}

{{< shloka  link="Shloka7" >}}

मत्तः परतरं नान्यत्किञ्चिदस्ति धनञ्जय ।  
मयि सर्वमिदं प्रोतं सूत्रे मणिगणा इव ॥७.७॥

{{< /shloka >}}


## श्लोक ७.८ {class="visually-hidden"}

{{< shloka  link="Shloka8" >}}

रसोऽहमप्सु कौन्तेय प्रभास्मि शशिसूर्ययोः ।  
प्रणवः सर्ववेदेषु शब्दः खे पौरुषं नृषु ॥७.८॥


{{< /shloka >}}


## श्लोक ७.९ {class="visually-hidden"}

{{< shloka  link="Shloka9" >}}

पुण्यो गन्धः पृथिव्यां च तेजश्चास्मि विभावसौ ।   
जीवनं सर्वभूतेषु तपश्चास्मि तपस्विषु ॥७.९॥

{{< /shloka >}}


## श्लोक ७.१० {class="visually-hidden"}

{{< shloka  link="Shloka10" >}}

बीजं मां सर्वभूतानां विद्धि पार्थ सनातनम्।  
बुद्धिर्बुद्धिमतामस्मि तेजस्तेजस्विनामहम् ॥७.१०॥

{{< /shloka >}}


## श्लोक ७.११ {class="visually-hidden"}

{{< shloka  link="Shloka11" >}}

बलं बलवतां चाहं कामरागविवर्जितम् ।  
धर्माविरुद्धो भूतेषु कामोऽस्मि भरतर्षभ ॥७.११॥

{{< /shloka >}}

## श्लोक ७.१२ {class="visually-hidden"}

{{< shloka  link="Shloka12" >}}

ये चैव सात्त्विका भावा राजसास्तामसाश्च ये ।  
मत्त एवेति तान्विद्धि नत्वहं तेषु ते मयि॥७.१२॥

{{< /shloka >}}

## श्लोक ७.१३ {class="visually-hidden"}

{{< shloka  link="Shloka13" >}}

त्रिभिर्गुणमयैर्भावैरेभिः सर्वमिदं जगत् ।  
मोहितं नाभिजानाति मामेभ्यः परमव्ययम्॥७.१३॥

{{< /shloka >}}


## श्लोक ७.१४ {class="visually-hidden"}

{{< shloka  link="Shloka14" >}}

दैवी ह्येषा गुणमयी मम माया दुरत्यया।  
मामेव ये प्रपद्यन्ते मायामेतां तरन्ति ते॥७.१४॥

{{< /shloka >}}


## श्लोक ७.१५ {class="visually-hidden"}

{{< shloka  link="Shloka14" >}}

न मां दुष्कृतिनो मूढाः प्रपद्यन्ते नराधमाः।  
माययापहृतज्ञाना आसुरं भावमाश्रिताः॥७.१५॥

{{< /shloka >}}


## श्लोक ७.१६ {class="visually-hidden"}

{{< shloka  link="Shloka16" >}}

चतुर्विधा भजन्ते मां जनाः सुकृतिनोऽर्जुन।  
आर्तो जिज्ञासुरर्थार्थी ज्ञानी च भरतर्षभ॥७.१६॥

{{< /shloka >}}


## श्लोक ७.१७ {class="visually-hidden"}
{{< shloka  link="Shloka17" >}}

तेषां ज्ञानी नित्ययुक्त एकभक्तिर्विशिष्यते ।  
प्रियो हि ज्ञानिनोऽत्यर्थमहं स च मम प्रियः ॥ ७.१७॥

{{< /shloka >}}


## श्लोक ७.१८ {class="visually-hidden"}
{{< shloka  link="Shloka18" >}}

उदाराः सर्व एवैते ज्ञानी त्वात्मैव मे मतम् ।  
आस्थितः स हि युक्तात्मा मामेवानुत्तमां गतिम् ॥७.१८॥

{{< /shloka >}}


## श्लोक ७.१९ {class="visually-hidden"}
{{< shloka  link="Shloka19" >}}

बहूनां जन्मनामन्ते ज्ञानवान्मां प्रपद्यते।  
वासुदेवः सर्वमिति स महात्मा सुदुर्लभः॥७.१९॥

{{< /shloka >}}


## श्लोक ७.२० {class="visually-hidden"}
{{< shloka  link="Shloka20" >}}

कामैस्तैस्तैर्हृतज्ञानाः प्रपद्यन्तेऽन्यदेवताः।   
तं तं नियममास्थाय प्रकृत्या नियताः स्वया॥ ७.२०॥

{{< /shloka >}}


## श्लोक ७.२१ {class="visually-hidden"}
{{< shloka  link="Shloka21" >}}

यो यो यां यां तनुं भक्तः श्रद्धयार्चितुमिच्छति ।  
तस्य तस्याचलां श्रद्धां तामेव विदधाम्यहम् ॥७.२१॥

{{< /shloka >}}

## श्लोक ७.२२ {class="visually-hidden"}
{{< shloka  link="Shloka22" >}}

स तया श्रद्धया युक्तस्तस्याराधनमीहते ।  
लभते च ततः कामान्मयैव विहितान् हि तान् ॥७.२२॥

{{< /shloka >}}

## श्लोक ७.२३ {class="visually-hidden"}
{{< shloka  link="Shloka23" >}}

अन्तवत्तु फलं तेषां तद्भवत्यल्पमेधसाम् ।  
देवान्देवयजो यान्ति मद्भक्ता यान्ति मामपि ॥७.२३॥

{{< /shloka >}}

## श्लोक ७.२४ {class="visually-hidden"}
{{< shloka  link="Shloka24" >}}

अव्यक्तं व्यक्तिमापन्नं मन्यन्ते मामबुद्धयः।  
परं भावमजानन्तो ममाव्ययमनुत्तमम्॥७.२४॥

{{< /shloka >}}

## श्लोक ७.२५ {class="visually-hidden"}
{{< shloka  link="Shloka25" >}}

नाहं प्रकाशः सर्वस्य योगमायासमावृतः।  
मूढोऽयं नाभिजानाति लोको मामजमव्ययम्॥७.२५॥

{{< /shloka >}}

## श्लोक ७.२६ {class="visually-hidden"}
{{< shloka  link="Shloka26" >}}

वेदाहं समतीतानि वर्तमानानि चार्जुन।  
भविष्याणि च भूतानि मां तु वेद न कश्चन॥७.२६॥

{{< /shloka >}}

## श्लोक ७.२७ {class="visually-hidden"}
{{< shloka  link="Shloka27" >}}

इच्छाद्वेषसमुत्थेन द्वन्द्वमोहेन भारत।  
सर्वभूतानि संमोहं सर्गे यान्ति परन्तप॥७.२७॥

{{< /shloka >}}

## श्लोक ७.२८ {class="visually-hidden"}
{{< shloka  link="Shloka28" >}}

येषां त्वन्तगतं पापं जनानां पुण्यकर्मणाम्।  
ते द्वन्द्वमोहनिर्मुक्ता भजन्ते मां दृढव्रताः॥७.२८॥

{{< /shloka >}}

## श्लोक ७.२९ {class="visually-hidden"}
{{< shloka  link="Shloka29" >}}

जरामरणमोक्षाय मामाश्रित्य यतन्ति ये।  
ते ब्रह्म तद्विदुः कृत्स्नमध्यात्मं कर्म चाखिलम् ॥७.२९॥

{{< /shloka >}}

## श्लोक ७.३० {class="visually-hidden"}
{{< shloka  link="Shloka30" >}}

साधिभूताधिदैवं मां साधियज्ञं च ये विदुः।  
प्रयाणकालेऽपि च मां ते विदुर्युक्तचेतसः॥७.३०॥

{{< /shloka >}}

</br>  

##### ॐ तत्सदिति श्रीमद्गगवद्वीतासूपनिषत्सु ब्रह्मविद्यायां योगशास्त्रे श्रीकृष्णार्जुनसंवादे  ज्ञानविज्ञानयोगो नाम सप्तमोऽध्यायः ॥ ६ ॥

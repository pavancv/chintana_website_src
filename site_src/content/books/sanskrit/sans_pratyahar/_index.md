# Pratyaharas  

These shortened names given to a group of letter
(abbreviations).

There are 44 pratyaharas.

The स्थान of each alphabet is per table below

{{< table "table  table-striped table-hover table-bordered" >}}


|**स्वर** |	**व्यञ्जन**     |		|**स्थान**|
| :---: | :----      | :----   | :----|
|अ	| क	ख	ग	घ	|	ह	|`कण्ठ`|
|इ	| च	छ	ज	झ	|	य	श | `तालु`| 
|ऋ | ट	ठ	ड	ढ	 |	र	ष	| `मूर्धा`|
|लृ	| त	थ	द	ध    |	ल	स	| `दन्त`|
|उ	|	प	फ	ब	भ	|			|`ओष्ट`|
|   |					 |			|	   |
|	|	ञ 	म 	ङ 	ण	न|		|`नासिक`|
|ए	ऐ|					 |		|`कण्ठ + तालु`|
|ओ	औ|					|      |`कण्ठ+ ओष्ट`|
|		|				|	व	|`दन्त +  ओष्ट` |


{{</table>}}
